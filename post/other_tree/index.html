打印目录结构

2019-11-19

mac windows 打印目录结构

1. tree 安装

brew install tree

2. 常用命令

命令 意义
tree -d 只显示文件夹
ree -L n 显示项目的层级。n表示层级数。
tree -a 查看某个文件夹下的所有文件
tree -I pattern 过滤不想要显示的文件或者文件夹
ree > tree.md 将项目结构输出到tree.md这个文件

3.示例

# 显示某个项目下2层的所有文件结构，同时又过滤node_modules文件夹
tree -L 2 -I "node_modules"
  • 输出结果如下：
  .
  ├── backend
  │   ├── backend
  │   ├── manage.py
  │   ├── odds
  │   └── templates
  ├── frontend
  │   ├── README.md
  │   ├── build
  │   ├── config
  │   ├── index.html
  │   ├── package.json
  │   ├── src
  │   └── static
  └── tree.md
